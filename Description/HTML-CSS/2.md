## 📘 WireFrame

> 🚩 **WireFrame이란?

웹 or 앱의 인터페이스를 시각적으로 묘사한것, 어떤 목적을 가진 프로그램인지 알 수 있게 설계하는것이 중요함.

<br>

> 🚩 **Mock-up**

**웹 or 앱 에서의 목업은 기능적으로는 동작하지 않지만, 실제 제품이 작동하는 모습과 동일하게 HTML,CSS 작성하는것**

<br>

> 🚩 **HTML로 웹 앱 구조잡기**

**1. 큰틀에서 영역 나누기**

- r , w 영역 나누기
- 태그별로 영역 나누기
